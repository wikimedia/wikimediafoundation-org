== PP Collaborative Editing Pack for WordPress ==
	An extension to the PublishPress Permissions Pro plugin, an advanced content permissions engine and management interface. 
	
	Supports content-specific editing permissions, term assignment and page parent limitations. In combination with other extensions, supports custom moderation statuses and Revisionary.
